# Hydra
Nesse repositório abordarei mais a respeito do Hydra, que é uma software conhecido em segurança da informação e pentest. Não é exatamente uma ferramenta "hacker", mas sim uma ferramenta que pode ser usada por profissionais para testar a segurança de sistemas e redes e analisar suas vulnerabilidades.
